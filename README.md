# Cracking the Code

TypeScript solutions for [Cracking the Coding Interview](https://www.crackingthecodinginterview.com/) (6th Edition), mapped to the corresponding [LeetCode LCCI](https://leetcode.cn/problemset/lcci/) problems.

## Setup

```bash
npm install       # Install dependencies
npm test          # Run all tests
npm run build     # Compile TypeScript
npm run lint      # Lint source files
npm run format    # Format with Prettier
```

## Problem Mapping

Each exercise maps to a chapter and problem in the LCCI collection. Problems marked with \* are book-only (no LCCI counterpart).
